# Contributing

Thanks for wanting to add to this list.

## Rules

1. Submissions must be a real, working tool, library, or community.
2. Free tools and open-source projects always get priority listing.
3. One-line description, no marketing fluff.
4. Alphabetical within each section unless ordered by quality.

## How to add

1. Fork
2. Edit README.md
3. Open PR with a one-line summary of what you added

PRs that add SEO spam, affiliate links, or duplicate listings will be closed.
